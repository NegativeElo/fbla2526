General Objective

Your goal is to explore five different careers paths: Computer Scientist, Chemist, Stock Trader, Judge, and Lawyer. Complete the challenge goal in each game to certify yourself in that career. Higher scores will get you more points. Complete the game by certifying yourself in all 5 careers or by achieving a total score of 100 or more!

How to Play

  • Computer Scientist: Drag and drop code blocks to navigate the player (the red ball) to the green ball. Avoid obstacles and don't crash into walls.
        Challenge Goal: Collect the green ball at least 5 times in 2 minutes

  • Stock Trader: Buy low and sell high. Use Limit Orders to automate your trades while the market fluctuates.
        Challenge Goal: Increase your net worth by $50,000 in 5 minutes

  • Judge: Read the case files carefully. Deliver your verdict based on the evidence presented before time runs out.
        Challenge Goal: Become a Supreme Court Justice

  • Chemist: Select the elements that make up chemicals.
        Challenge Goal: Match 20 compounds with their chemical makeup correctly

  • Lawyer: Watch the actions of the opposing counsel. Make the correct objection.
        Challenge Goal: Become a Managing Partner

Controls

All games are playable via the mouse. However, certain games also have keys that perform the same action. You can press Enter to bypass intro screens and ESC at any time to quit a game and return to the main hub.

Technical Implementation (For Judges)

This simulation is built using a modern web stack to ensure cross-platform compatibility:

  • Programming Languages: HTML5, CSS3, and Vanilla JavaScript (ES6+).
  • Libraries: Google Blockly API (for the Computer Scientist visual programming logic).
  • Graphics: HTML5 Canvas API for real-time rendering of game grids and stock market charts.
  • State Management: Real-time data persistence using localStorage API to track scores and completion status across browser sessions.
